# Ultrack active learning + visualization

This application is our first driving example of an active learning application.

It is a prototype that is intended to inform technical design choices around active learning applications and scientific imaging visualization.

It is intended for internal demos and testing, but not for public dissemination.

It uses React as a front-end framework and also uses components from CZI's Science Design System.

## Development

To setup a development environment run

```shell
npm install
```

To start a local server for development run

```shell
npm run dev
```
