# OMEZarr 0.4 type definitions

These files were generated by the [quicktype](https://github.com/glideapps/quicktype) utility.

Regenerating these files looks something like this:
```
npm install -g quicktype
git clone git@github.com:ome/ngff.git /tmp/ngff
export CI=true  # Workaround for https://github.com/glideapps/quicktype/issues/2613
for i in $(ls /tmp/ngff/0.4/schemas/strict*.schema | xargs basename | sed 's/strict_//' | cut -d '.' -f 1); do quicktype --src /tmp/ngff/0.4/schemas/strict_$i.schema --src-lang schema --lang typescript --prefer-types -o $i.ts; done
```
