# OMEZarr 0.4 type definitions

These files were generated by the [json-schema-to-typescript](https://github.com/bcherny/json-schema-to-typescript) utility.

Regenerating these files looks something like this:

```
npm install -g json-schema-to-typescript
git clone git@github.com:ome/ngff.git /tmp/ngff
cp /tmp/ngff/0.4/schemas/*.schema ./schemas
for i in `ls ./schemas/*.schema | grep -v strict | xargs basename | cut -d '.' -f 1`; do json2ts schemas/$i.schema > $i.ts; done
```
