# Zarr type definitions

These files were generated by the [json-schema-to-typescript](https://github.com/bcherny/json-schema-to-typescript) utility.

Regenerating these files looks something like this:

```
npm install -g json-schema-to-typescript
git clone git@github.com:ome/ngff.git
cd ngff/0.4/schemas
for i in for i in `ls *.schema | grep -v strict | cut -d '.' -f 1`; do json2ts $i.schema > $i.ts; done
cp *.ts /path/to/this/directory
```
