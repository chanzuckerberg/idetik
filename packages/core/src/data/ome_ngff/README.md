# OME-Zarr type definitions

The files in each versioned sub-directory (e.g. 0.4) were generated for use with [zod](https://zod.dev/).

They were generated using the `@apidevtools/json-schema-ref-parser` and `json-schema-to-zod` Node packages, which should be installed as dev dependencies.

The files can be regenerated by running the following command with the appropriate version number and schema names. For example, the following command was used to generate the v0.4 schemas that Idetik uses.

```bash
./make_zod_types 0.4 image plate well
```

Some small manual modifications were made to the generated files to account for perceived ambiguities in some versions of the OME-Zarr specification and their common implementations.
