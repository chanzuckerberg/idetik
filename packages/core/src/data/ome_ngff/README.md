# OME-Zarr type definitions

The files in each version sub-directory were generated for use with [zod](https://zod.dev/).

They were generated using the `@apidevtools/json-schema-ref-parser` and `json-schema-to-zod` Node packages, which can be installed globally by running the following command.

```bash
npm install --save-dev @apidevtools/json-schema-ref-parser json-schema-to-zod
```

The files can be regenerated by running the following command with the appropriate version number.

```bash
./make_zod_types 0.4
```

Some small manual modifications were made to the generated files to account for perceived ambiguities in some versions of the OME-Zarr specification.
